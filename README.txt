# People Manager - Управление пользователями

Простое desktop-приложение для управления списком пользователей с графическим интерфейсом на Tkinter и базой данных PostgreSQL.

## 📋 О проекте

Приложение позволяет:
- ✅ Добавлять новые записи (имя + информация)
- ✅ Удалять выбранные записи
- ✅ Очищать всю таблицу
- ✅ Автоматическое сохранение данных в PostgreSQL
- ✅ Удобный интерфейс с таблицей и скроллингом

## 🚀 Технологии

- **Python 3.8+**
- **Tkinter** - графический интерфейс
- **PostgreSQL** - база данных
- **psycopg2** - драйвер для PostgreSQL
