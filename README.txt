================================================================================
PEOPLE MANAGER - Управление пользователями
================================================================================

Простое desktop-приложение для управления списком пользователей с графическим 
интерфейсом на Tkinter и базой данных PostgreSQL.

================================================================================
О ПРОЕКТЕ
================================================================================

Приложение позволяет:
- Добавлять новые записи (имя + информация)
- Удалять выбранные записи
- Очищать всю таблицу
- Автоматическое сохранение данных в PostgreSQL
- Удобный интерфейс с таблицей и скроллингом

================================================================================
ТЕХНОЛОГИИ
================================================================================

- Python 3.8+
- Tkinter - графический интерфейс
- PostgreSQL - база данных
- psycopg2 - драйвер для PostgreSQL

================================================================================
УСТАНОВКА
================================================================================

1. Клонирование репозитория:

   git clone https://github.com/yourusername/people-manager.git
   cd people-manager

2. Установка зависимостей:

   pip install psycopg2

   Если не работает:
   pip install psycopg2-binary

3. Настройка PostgreSQL:

   Проверка версии:
   psql --version

   Проверка статуса (Linux):
   sudo systemctl status postgresql

   Проверка статуса (MacOS):
   brew services list | grep postgres

   Проверка статуса (Windows):
   sc query postgresql

4. Настройка подключения к БД:

   Откройте файл db.py и измените пароль на свой:

   self.config = {
       'host': 'localhost',
       'port': '5432',
       'user': 'postgres',     # ИЗМЕНИТЕ на вашего пользователя
       'password': 'staff',    # ИЗМЕНИТЕ на ваш пароль
       'dbname': 'postgres'
   }

5. Запуск приложения:

   python manager.py

6. Первый запуск:

   При первом запуске автоматически создадутся:
   - База данных 'my_new_database'
   - Таблица 'users'
   - Загрузятся существующие данные из БД

================================================================================
ИСПОЛЬЗОВАНИЕ
================================================================================

Добавление записи:
   1. Введите имя в поле "Имя"
   2. Введите информацию в поле "Информация о записи"
   3. Нажмите "Добавить строку" или клавишу Enter

Удаление записи:
   1. Выделите строку в таблице (кликните по ней)
   2. Нажмите "Удалить выделенную строку"
   3. Подтвердите удаление

Очистка всей таблицы:
   1. Нажмите "Очистить таблицу"
   2. Подтвердите действие

================================================================================
ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
================================================================================

ПРОБЛЕМА: psycopg2.OperationalError: FATAL: password authentication failed
РЕШЕНИЕ: Проверьте пароль в файле db.py

ПРОБЛЕМА: permission denied to create database
РЕШЕНИЕ: Дайте пользователю права:
         ALTER USER postgres WITH CREATEDB;

ПРОБЛЕМА: ModuleNotFoundError: No module named 'psycopg2'
РЕШЕНИЕ: pip install psycopg2-binary

ПРОБЛЕМА: База данных не создается
РЕШЕНИЕ: Убедитесь что у пользователя есть права на создание БД

================================================================================
ПРОВЕРКА ПОДКЛЮЧЕНИЯ
================================================================================

Создайте файл test_connection.py:

import psycopg2

try:
    conn = psycopg2.connect(
        host='localhost',
        port='5432',
        user='postgres',
        password='ВАШ_ПАРОЛЬ',
        dbname='postgres'
    )
    print("Подключение успешно!")
    conn.close()
except Exception as e:
    print(f"Ошибка: {e}")

Запустите: python test_connection.py
