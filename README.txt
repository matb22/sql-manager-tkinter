# Database Table Manager (Tkinter + PostgreSQL)

Простое десктопное приложение на Python для управления записями в базе данных PostgreSQL. Интерфейс построен на стандартной библиотеке tkinter, а хранение данных реализовано через драйвер psycopg2.

## Функционал

- Добавление записей: Ввод имени и дополнительной информации с сохранением в БД.
- Просмотр: Интерактивная таблица (Treeview) с поддержкой вертикальной и горизонтальной прокрутки.
- Удаление: Возможность удалить выбранную строку из интерфейса и базы данных одновременно.
- Очистка: Быстрое удаление всех данных (сброс таблицы).
- Автоматизация: База данных и таблица создаются автоматически при первом запуске приложения.

## Технологии

- Python 3.x
- Tkinter (Графический интерфейс)
- PostgreSQL (Система управления базами данных)
- psycopg2 (Адаптер БД для Python)

## Установка и запуск

### 1. Требования
Убедитесь, что у вас установлен и запущен сервер PostgreSQL. По умолчанию приложение использует следующие настройки подключения (их можно изменить в db.py):
- Host: localhost
- User: postgres
- Password: staff
- Database: postgres (техническая БД для инициализации)

### 2. Клонирование репозитория
git clone https://github.com/ваш-логин/название-репозитория.git
cd название-репозитория

### 3. Установка зависимостей
Установите драйвер для работы с базой данных:
pip install psycopg2-binary

### 4. Запуск приложения
python manager.py

## Структура проекта

* manager.py — Главный файл приложения: содержит логику интерфейса (GUI) на Tkinter и обработку событий.
* db.py — Слой работы с данными: содержит класс myDB для выполнения SQL-запросов (создание БД, вставка, удаление, выборка).

## Важное примечание
Приложение автоматически создает базу данных с именем my_new_database. Если вы хотите использовать существующую базу или изменить учетные данные, отредактируйте метод __init__ в файле db.py.
